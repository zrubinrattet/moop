moop

A modern desktop image optimizer.

Allowing you to moo-ve faster with lighter images.

Download for macOS (Apple Silcon) View on Github

After copying the app to your Applications folder, run the following in Terminal to remove the app from Apple's Quarantine:

xattr -cr /Applications/moop.app

Features

Batch Jobs

Optimize many images in one run.

Drag & Drop

Drop files in and start instantly.

Format Choice

Output as WebP, PNG, or JPEG.

Quality Tuning

Balance file size and visual fidelity.

Smart Resize

Scale down while keeping proportions.

Animation Ready

Preserve animation for GIF and WebP.

Many Languages

Use moop in your preferred language.

Local Only

Processing stays on your device.
